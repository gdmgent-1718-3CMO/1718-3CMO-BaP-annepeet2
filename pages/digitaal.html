Website

Deze responsive website is op maat gemaakt voor deze twirlclub. Het bestaat uit een one-page met de typische Attitude elementen. Je kan deze website bezoeken via www.attitudetwirlclub.be.

Nieuwsbrief

De digitale nieuwsbrief is gemaakt via MailChimp. Hierdoor is er een beperking aan lettertypes. Gebruik daarom de lettertypes Helvetica voor de titles en Times New Roman voor de doorlopende tekst. Voor het juiste gebruik leest u best even de typografie pagina op deze website.
